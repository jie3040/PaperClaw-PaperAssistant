# Surveyor 工作流

## Mode A（从零开始）
全面检索：主题核心方法、各子方向、近期进展、综述论文。

## Mode B（结果先行）
Leader 的任务指令中会包含 "Mode B" 关键词和 `materials_summary.md` 路径。

### 聚焦 Related Work 检索
1. 读取 `SHARED/user_materials/materials_summary.md` 了解用户的方法
2. 重点检索：
   - 与用户方法最相关的已有工作
   - 用户方法所基于/改进的基础方法
   - 同领域的竞争方法（用于实验对比 baseline）
   - 用户使用的数据集/评价指标的原始论文
3. 其余检索策略与 Mode A 相同
