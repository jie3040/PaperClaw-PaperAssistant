# 文献调研精简摘要 — Project 5

## 主题：Zero-shot Fault Diagnosis + Diffusion Model + Language Model
## 文献总数：58篇（2020-2026，重点2023-2026）

---

## 一、研究现状概览

### 1. 零样本故障诊断（23篇）
- **经典方法**：属性嵌入+映射（FDAT, SCE）、度量学习、注意力网络
- **生成式方法**：GAN系（FAGAN, SRWGAN, VAEGAN-AR, FREE）、CycleGAN-SD
- **核心挑战**：域偏移（DSP）、复合故障语义建模、跨域泛化
- **关键趋势**：物理知识融合、知识空间共享缓解域偏移

### 2. 扩散模型+故障诊断（16篇）
- **数据增强**：DDPM比GAN更稳定，无模式崩塌（文献24-32）
- **零样本结合**：扩散+ZSL是新兴方向，已有3篇TIM论文（文献37-39）
  - [37] Diffusion+多标签网络→广义零样本复合故障诊断（IEEE TIM 2024）
  - [38] 增强扩散模型→高铁转向架GZSL（IEEE TIM 2024）
  - [39] 扩散编码+软语义嵌入→零样本诊断（EAAI 2025）
- **领域泛化**：扩散增强提升跨域能力（文献33-36）

### 3. CLIP/视觉-语言模型（7篇）
- CLIP用于零样本工业缺陷检测（文献42-44）
- CLIP+时频图用于轴承故障迁移学习（文献41）
- FaultGPT：CLIP ViT-L/14 + 提示学习→故障诊断QA（文献45）
- **关键发现**：CLIP的文本编码器可替代传统二值属性（手工定义），提供更丰富的语义空间

### 4. LLM/基础模型（11篇）
- FD-LLM：多模态对齐+LLM微调→零样本故障分类（文献47）
- UniFault：90亿数据点预训练→跨域泛化基础模型（文献51）
- 多模态大模型LMM-FD：CLIP对齐+GNN→零样本诊断知识三元组（文献56）
- **关键发现**：LLM的文本理解能力可用于自然语言故障描述→语义嵌入

### 5. 域偏移与生成式方法
- CycleGAN-SD：语义距离+循环一致性→域间故障转换（范例论文2）
- DP-CDDPM-AC：双路径扩散+属性一致性→鲁棒零样本诊断（范例论文1）

## 二、研究空白（创新点机会）

1. **扩散模型+语言模型结合**：目前无文献同时使用diffusion model和language model进行零样本故障诊断
2. **语言模型替代手工属性**：现有ZSL方法均依赖专家手工定义二值属性，CLIP/LLM的文本编码可自动生成更丰富语义
3. **文本引导的扩散生成**：类似Stable Diffusion的text-to-image，可设计text-to-fault-feature的生成范式
4. **自然语言故障描述**：用LLM将专家故障描述转化为连续语义向量，替代离散二值属性
5. **跨模态对齐**：CLIP式对比学习对齐故障特征空间和文本描述空间
